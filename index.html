A CV-Builder Application, build with CSS3 and React. I learned the fundamentals of what React has to offer. such as, writing JSX code, working with components, passing data between components (props), useState and useEffect

A Tic-Tac-Toe Game to play with your friends. build using HTML5 CSS3 and JS

An online Etch-A-Sketch with high functionality, you can choose between colors, erase. and being able to choose your own grid-size. Built using HTML5, CSS3 and JS.
